# Docker
## 镜像
##### ubuntu
```sh
docker pull ubuntu
```
##### debian
```sh
docker pull debian
```
## 更新
##### update
```sh
apt-get update
```
## 安装软件
##### sudo
```sh
apt-get install sudo
```
##### curl
```sh
apt-get install curl
```

curl -k https://raw.iqiq.io/3wking/Synology/main/Docker/ssh.sh | bash
